# Detecting fake-news-brand-impact
NLP project to detect fake news impacting brands

# Project Overview
This project uses Natural Language Processing (NLP) and Machine Learning techniques to detect fake news impacting brands and analyze brand reputation impact.

# Features
- Fake vs Real news classification
- Sentiment analysis
- Brand impact analysis
- Data visualization
- Machine learning model comparison

## Technologies Used
- Python
- Pandas
- Scikit-learn
- NLTK
- Matplotlib
- Seaborn
- Google Colab

## Models Used
- Logistic Regression
- Naive Bayes

## Dataset
The dataset contains 1000 records with:
- News text
- Brand
- Label
- Sentiment score
- Brand reputation impact

# Project Workflow
Dataset → Data Preprocessing → TF-IDF Vectorization → Machine Learning Models → Prediction → Evaluation → Visualization

# How to Run the Project
1. Upload the dataset in Google Colab
2. Open the notebook file
3. Run all cells sequentially
4. Generate predictions and visualizations
5. Analyze model performance

# Key Insights
- Fake news is mostly associated with negative sentiment
- High-impact news is generally fake and harmful
- Certain brands are more frequently targeted
- NLP helps organizations monitor misinformation effectively

# Results
The models achieved strong performance in fake news classification with realistic accuracy metrics.
