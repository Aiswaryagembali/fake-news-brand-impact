# Detecting fake-news-brand-impact
NLP project to detect fake news impacting brands

## Project Overview
This project uses Natural Language Processing (NLP) and Machine Learning techniques to detect fake news impacting brands and analyze brand reputation impact.

## Features
- Fake vs Real news classification
- Sentiment analysis
- Brand impact analysis
- Data visualization
- Machine learning model comparison

## Technologies Used
- Python
- Pandas
- Scikit-learn
- NLTK
- Matplotlib
- Seaborn
- Google Colab

## Models Used
- Logistic Regression
- Naive Bayes

## Dataset
The dataset contains 1000 records with:
- News text
- Brand
- Label
- Sentiment score
- Brand reputation impact

## Results
The models achieved strong performance in fake news classification with realistic accuracy metrics.
